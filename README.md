# 3D Cube Counter (Würfelzähler)

This is a simple, single-page web application built with HTML, CSS, and JavaScript that visually represents stacks of cubes in a 3D space and automatically counts them.

Dies ist eine einfache, einseitige Webanwendung, die mit HTML, CSS und JavaScript erstellt wurde. Sie visualisiert Würfelstapel in 3D und zählt diese automatisch.

## Features

*   **3D Visualization:** Displays a 3x3 grid with randomly generated stacks of cubes in a draggable 3D view.
*   **2D Blueprint:** Shows a corresponding 2D grid that gets filled with the cube counts.
*   **Automatic Solving:** An animated process that iterates through each stack, counts the cubes, and calculates the total sum.
*   **Interactive View:**
    *   Rotate the 3D scene by clicking and dragging.
    *   Zoom in and out using the mouse wheel.
    *   Reset the view to its original position.

## How to Use

1.  Open the `index.html` file in a web browser.
2.  A random arrangement of cubes will be generated.
3.  Click the "**Lösen & Zählen**" (Solve & Count) button to start the counting animation.
4.  Watch as the application highlights each stack, updates the 2D blueprint, and shows the running total.
5.  Click the "**Neue Aufgabe**" (New Task) button to generate a new set of cubes.
6.  Use your mouse to interact with the 3D view.

## Technology

This project is a self-contained application using only vanilla front-end technologies:

*   **HTML5**
*   **CSS3** (including 3D transforms and animations)
*   **JavaScript** (ES6+)
